# 📊 EDA & Business Intelligence Dashboard
## Overview
This project focuses on Exploratory Data Analysis (EDA), SQL-based Business Intelligence, and Power BI Dashboard Development to analyze career growth, popularity trends, political activities, and family relationships.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL (SQLite)
- Power BI
## Analysis Performed
- Descriptive Statistics
- Univariate Analysis
- Correlation Analysis
- Multivariate Analysis
- SQL Business Queries
- Trend Analysis
## Dashboard Highlights
- Average Popularity Score
- Total Followers
- Total Movies
- Political Events Count
- Genre Distribution
- Family Relationship Analysis
- Popularity vs Followers Trends
## Key Insights
- Popularity and followers show a strong positive relationship.
- Action and Drama genres dominate the film career dataset.
- Popularity has increased steadily over time.
- Political activities show notable trends across different years.
## Outcome
This project strengthened my skills in Data Analysis, SQL, Data Visualization, Business Intelligence, and Dashboard Design.
